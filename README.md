# Tarok-belezka
Program bo v pomoč beleženju točk pri dolgotrajnih igrah taroka.
Program deluje tako, da razen  v primeru vnašanja razlike igre vpisujete števila pred izbiro programa.
Rad bi se pohvalil, da program zna računati po uradnem točkovanju. Za ekstravagantneže sem dodal tudi igro pikolo-berač.
Ko vpisujete ime vaše mize, bi vas opozoril, da ime zelo dobro zapomnete, saj deluje tudi kot geslo. Vsakršna razlika (male, velike črke, posebni znaki, presledki) dajejo mizi edinstveno ime, ki obenem varuje in hrani vaše podatke pred možnostmi vdora.
Upam, da bo komu prišlo prav.
